## 1. 数组、链表、跳表基本原理

#### 时间复杂度

>数组

```
- 时间复杂度
prepend O(1)
append O(1)
lookup O(1)
insert O(n)
delete O(n)
```

>链表

```
prepend O(1)

```

>跳表

```
O(logn)
```


## 2. 栈、队列、优先队列和双端队列的特性


>栈（先进后出）

```
添加、删除 O(1)，查询 O(n)
```


>队列（先进先出）

```
添加、删除 O(1)，查询 O(n)
```

>双端队列（头和尾都可以进出）

```
添加、删除 O(1)，查询 O(n)
```


>优先队列（元素被赋予优先级）

```
添加 O(1)，获取 O(n)
```
